# SUMOBOT project - DRAGON NEIGE - FR
La conception, la modélisation 3D et la programmation d’un robot SUMOBOT dans le cadre d’un concours universitaire organisé en partenariat avec une école maternelle. Le concept et les caractéristiques opérationnelles du robot ont été déterminés par un groupe de six enfants de maternelle.

# SUMOBOT project - DRAGON SNOW - EN
Design, 3D modeling, and programming of a SUMOBOT robot as part of a university competition carried out in partnership with a kindergarten. The concept and operational features of the robot were determined by a group of six kindergarten children.
